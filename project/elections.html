What is a Social Bot?

Social media in recent years saw a surge in use and exposure. While most users are controlled by human, some control by automate agent or human with robotic behavior.
Those users try to imitating humans to manipulate discussions, spread "Fake News" and divert an opinion

Data Collection

We collected post and comments of each candidate in the elections with a Crawler based on selenium.
We saved the data in JSON and uploaded it to AWS server.

In addition, we chose selectively group of users - randomly, comments number, elections activity and selected post - and manually scan the user to add features and ruling whatever the user is a bot.

Main Goal

Building a machine learning model to predict user/comment credibility - whatever his is a Social Bot (or sybil account)

Data Analysis

Using multiple data visualiztion tools such as python libaries - mathplotlib, seaborn and creating ElasticSearch dashboard to examine our data.

Classification System

We can group our features to x main classes:
Metadata, activity and user profile.

Feature Selection

We checked each feature correlation with eachother and our ruling. Also we tried to create new metrics from our data such as posts per comments, elctions active etc.
